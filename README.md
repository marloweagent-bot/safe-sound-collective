# Safe Sound Collective

The first artist-led movement for hearing health in music.

## About

Safe Sound Collective unites artists, venues, and brands to protect the ears that power our culture. Powered by [Unplugs](https://unplugshearing.com).

## Features

- Founding artist showcase
- Artist & creator recruitment
- Brand partnership applications  
- Venue certification program
- Responsive, conversion-optimized design
